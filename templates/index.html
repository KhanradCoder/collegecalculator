How It Works

CollegeCalculator AI uses admission data from CollegeData.com. Colleges with similar selectivity are group togther and trained through a deep neural network in tensorflow. Once the deep learning model is trained, a linear model is created for each college based on that college's average SAT, admission rate, etc. Because of this, CollegeCalculator is able to individualize deep learning models for each college. This method is not perfect, but fairly accurate given the lack of specific data on college admissions. If you would like to learn more about how CollegeCalculator works, please visit the github page where you can view the full code linked here.
